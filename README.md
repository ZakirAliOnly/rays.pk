# Rays Marketing Network — Official Website

> **"We Believe in Quality Services"**  
> Est. 2010 · Quetta, Pakistan

A fully static, single-page business website for **Rays Marketing Network** — a government-registered marketing, printing, and supply company based in Quetta, Pakistan.

---

## 🌐 Live Preview

Open `index.html` directly in any browser — no build step or server required.

To host on **GitHub Pages**:
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch / `/ (root)`
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

## ✨ Features

- **Hero section** with animated particle canvas and counter stats
- **Marquee strip** of service keywords
- **About** section with mission & vision cards
- **Services** grid (10 services)
- **Product Catalogue** with tabbed filter (School / Office / Print & Brand)
- **Clients** auto-scrolling carousel
- **Partners** auto-scrolling logo strip
- **Certificates** carousel with popup lightbox (click to enlarge)
- **Contact** section with registration details
- **WhatsApp floating button** for instant enquiries
- **Background music** toggle
- Fully **responsive** (mobile hamburger nav, mobile pill layout)
- Scroll-based **reveal animations**
